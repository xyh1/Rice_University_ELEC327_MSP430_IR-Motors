Welcome to Team MJX's ELEC 327 Tutorial on MSP430 IR Motor Control

This is a project tutorial and presentation for Rice University's ELEC 327 class taught by Dr. Kemere (@ckemere).
It will be going over how a MSP430G2553 reads and decodes IR signals and how it translates those signals into motor commands to drive a vehicle.
Commented code and project documentation is provided via the github repo page.
